# Versioning Rule

When making changes to the plugin (adding, modifying, or removing commands, skills, agents, or MCP servers), you MUST update the version in `.claude-plugin/plugin.json`.

Follow semantic versioning:
- **Patch** (1.0.X): Bug fixes, documentation updates
- **Minor** (1.X.0): New features, new commands/skills/agents
- **Major** (X.0.0): Breaking changes
